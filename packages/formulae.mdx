Homebrew formulae are package definitions that download and install command-line tools. The HashiCorp tap includes 32 formulae covering infrastructure automation, secrets management, service networking, access management, and platform tooling. Most formulae install the official, signed binary published by HashiCorp — identical to what you would download from [releases.hashicorp.com](https://releases.hashicorp.com). A small number of formulae (`copywrite`, `enos`) are maintained differently and fetch their binaries from GitHub releases.

## Install pattern

```bash
brew install hashicorp/tap/<product>
```

Always use the fully qualified `hashicorp/tap/` prefix to ensure you install from the official HashiCorp tap rather than another source.

## Architecture notes

<Warning>
  Some formulae do not support every architecture. Installing on an unsupported platform will fail. Check the tables above before installing on Linux ARM 32-bit or macOS Apple Silicon.
</Warning>

Products with restricted architecture support:

- **`consul-aws`** — not available on `darwin_arm64` or `linux_arm`
- **`consul-terraform-sync`** — not available on `darwin_arm64`
- **`nomad`**, **`nomad-enterprise`**, **`nomad-pack`**, **`levant`** — not available on `linux_arm`
- **`vault`**, **`vault-enterprise`**, **`vault-radar`** — not available on `linux_arm`
- **`vagrant`** (formula) — Linux `linux_amd64` only; use the [Vagrant cask](/packages/casks) on macOS
- **`copywrite`**, **`enos`** — not available on `linux_arm`
