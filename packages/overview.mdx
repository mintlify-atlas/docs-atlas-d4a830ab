The HashiCorp Homebrew tap provides official, signed binaries for HashiCorp products on macOS and Linux. You get the exact same binaries published to [releases.hashicorp.com](https://releases.hashicorp.com) — compiled with the Go version and build flags chosen by the teams who maintain each product. The tap contains two package types: **formulae** for CLI tools (available on both macOS and Linux) and **casks** for macOS GUI applications.

## Supported architectures

The tap publishes binaries for the following platform and architecture combinations. Not every formula supports every architecture — see the [formulae page](/packages/formulae) for per-product details.

| Architecture | Identifier | Notes |
|---|---|---|
| macOS Intel | `darwin_amd64` | All formulae and casks |
| macOS Apple Silicon | `darwin_arm64` | All formulae and casks (most products) |
| Linux x86-64 | `linux_amd64` | All formulae |
| Linux ARM 32-bit | `linux_arm` | Select formulae only |
| Linux ARM 64-bit | `linux_arm64` | Most formulae |

## GA releases only

<Note>
  Only generally available (GA) releases are published to this tap. Alpha, beta, and release candidate (prerelease) versions will not appear here.
</Note>

## Searching and inspecting packages

After adding the tap, you can search across all available packages:

```bash
brew search hashicorp
```

To view details about a specific package — including its current version, dependencies, and install location — use `brew info` with the fully qualified name:

```bash
brew info hashicorp/tap/vault
```

You must always use the `hashicorp/tap/` prefix when referring to packages from this tap outside of `brew search`, since the same product name may exist in other taps.
