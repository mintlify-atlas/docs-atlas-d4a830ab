The HashiCorp Homebrew Tap is the official source for installing HashiCorp products via Homebrew on macOS and Linux. Unlike community-maintained taps, this tap distributes the exact, byte-for-byte binaries that HashiCorp publishes — signed by HashiCorp, built with deliberate Go version and build flag choices, and automatically updated when new releases ship.

## Install a package

Add the tap and install any HashiCorp product with a single command:

```bash
brew install hashicorp/tap/terraform
```

Or add the tap first to make all packages searchable:

```bash
brew tap hashicorp/tap
brew install hashicorp/tap/vault
```

## Get started

<Steps>
  <Step title="Install Homebrew">
    If you don't have Homebrew installed, get it from [brew.sh](https://brew.sh):

    ```bash
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    ```
  </Step>
  <Step title="Add the tap">
    Add the HashiCorp tap to your Homebrew installation:

    ```bash
    brew tap hashicorp/tap
    ```
  </Step>
  <Step title="Install a product">
    Install any HashiCorp product using the fully qualified name:

    ```bash
    brew install hashicorp/tap/terraform
    ```
  </Step>
  <Step title="Verify the installation">
    Confirm the binary is installed and check its version:

    ```bash
    terraform --version
    ```
  </Step>
</Steps>

<Note>
  Prereleases — including alphas, betas, and release candidates — are not available in this tap. Only generally available (GA) releases are published here.
</Note>
