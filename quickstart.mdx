This guide walks you through adding the HashiCorp Homebrew Tap to your system and installing your first HashiCorp product. By the end you will have Homebrew installed, the `hashicorp/tap` tap registered, and a working HashiCorp CLI tool ready to use.

## Prerequisites

- macOS or Linux
- A terminal with internet access

If you already have Homebrew installed, you can skip directly to step 2.

## Installation steps

<Steps>
  <Step title="Install Homebrew">
    If Homebrew is not yet on your system, install it by running the official install script:

    ```bash
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    ```

    Follow the prompts. On Linux, the installer will also print post-install instructions to add Homebrew to your shell's `PATH` — run those commands before continuing.
  </Step>

  <Step title="Add the HashiCorp tap">
    Register the HashiCorp tap with Homebrew. This makes all formulae in the tap available in `brew search` results and lets you refer to them by their short names in subsequent commands.

    ```bash
    brew tap hashicorp/tap
    ```

    <Note>
      To clone the tap via SSH instead of HTTPS, pass the full repository URL:

      ```bash
      brew tap hashicorp/tap https://github.com/hashicorp/homebrew-tap
      ```
    </Note>
  </Step>

  <Step title="Install a product">
    Install Terraform as your first HashiCorp product:

    ```bash
    brew install hashicorp/tap/terraform
    ```

    Always use the fully qualified name `hashicorp/tap/<product>` when installing from an external tap. Homebrew requires this prefix outside of search to avoid ambiguity with packages that may exist in other taps or in `homebrew-core`.
  </Step>

  <Step title="Verify the installation">
    Confirm that Terraform installed correctly and is on your `PATH`:

    ```bash
    terraform --version
    ```

    You should see output like `Terraform v1.x.x` confirming the installed version.
  </Step>
</Steps>

## Install without adding the tap first

You do not have to register the tap before installing a package. Homebrew accepts the fully qualified name directly and adds the tap automatically:

```bash
brew install hashicorp/tap/vault
```

This is equivalent to running `brew tap hashicorp/tap` followed by `brew install hashicorp/tap/vault`. Adding the tap explicitly first is useful if you want packages to appear in `brew search` results before you install anything.
