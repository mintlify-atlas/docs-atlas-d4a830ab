The HashiCorp Homebrew Tap (`hashicorp/tap`) is the official repository HashiCorp maintains for distributing its products through Homebrew. Every binary available here is identical, byte-for-byte, to the release published on [releases.hashicorp.com](https://releases.hashicorp.com), signed by HashiCorp, and updated automatically as new versions ship. If you want the authoritative build of any HashiCorp tool on macOS or Linux, this tap is the right place to get it.

## What is Homebrew?

Homebrew is the most widely used package manager for macOS, and it also runs on Linux. It lets you install, update, and remove command-line tools and desktop applications with a single command. You can learn more at [brew.sh](https://brew.sh).

## What is a tap?

A tap is a third-party Homebrew repository that provides additional formulae and casks beyond what ships in Homebrew's built-in catalog (`homebrew-core`). When you add a tap, Homebrew gains access to everything that tap publishes. You reference packages from an external tap using a fully qualified name that includes the tap owner and repository, for example `hashicorp/tap/terraform`.

## Why use this tap instead of homebrew-core?

Homebrew's community-maintained `homebrew-core` tap compiles HashiCorp software from source on Homebrew's own infrastructure. That approach is fine for many use cases, but it means the resulting binary may differ from the one HashiCorp published. The HashiCorp Homebrew Tap is different in several important ways:

- **Official, signed binaries.** macOS binaries are signed by HashiCorp and can be verified using the instructions on the [HashiCorp Security page](https://www.hashicorp.com/security#code-signature-verification).
- **Exact byte-for-byte match.** The binaries distributed here are the same artifacts published to releases.hashicorp.com — nothing is recompiled.
- **Deliberate build choices.** HashiCorp's engineering teams select the Go version and build flags with support and security in mind. Those choices are reflected in every binary here.
- **Automated updates.** Formula updates are triggered automatically when a new release ships, so you get access to new versions quickly.

## Architecture overview

This tap contains two types of packages:

- **Formulae** are CLI tools installed to your `PATH`. Examples include `terraform`, `vault`, `consul`, `nomad`, and `packer`. Formulae work on both macOS and Linux.
- **Casks** are macOS GUI applications distributed as `.app` or `.pkg` bundles. The tap currently includes `hashicorp-boundary-desktop` and `hashicorp-vagrant`. Casks are macOS-only.
